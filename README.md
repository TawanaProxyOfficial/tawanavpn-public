# Tawanavpn / Tawana Proxy — Public Documentation | مستندات عمومی

This repository is **public / sanitized / non-operational**, providing the bilingual (Persian/English) Tawanavpn roadmap materials. It organizes the full FA/EN articles, publication checklist, metadata, license, and a GitHub Pages microsite for sharing via Figshare/Zenodo or similar research archives.
